## B問題

### 解法メモ
✖　配列の初めの2つから, 比を取り,他の要素同士の比と一致するか判定.
　　☝この方法だと,分数の比較に誤差がでてきてしまうため、WA

### 模範解答
- 連続する2項の比が等しいか判定.判定の仕方として、
```Python
a[i]*a[i+2]!=a[i+1]*a[i+1] #正しい:整数で比較.

a[i+1]/a[i] != a[i+2]/a[i+1] #誤り:小数で比較.誤差が生まれやすい.
```


### 振り返り
**🔴 小数同士の比較は誤差が生まれやすい**
- 整数同士でできる方法は無いか模索する.
- もっと一般化出来ないか考える.
