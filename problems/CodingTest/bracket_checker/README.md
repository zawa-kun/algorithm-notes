# 括弧の対応を調べる
## 問題
入力："(",")","{",","}","[","]"で構成された文字列.
出力：括弧が対応しているか(Yes/No)
### 入力例
```shell
()          出力：Yes
{}{}        出力：Yes
{]}[]       出力：No
({)}        出力：Yes
(((()){}))  出力：Yes
```
## 考え方
- 一つ目から見ていって,もし,始まり括弧"(","{","["があったら、
- checkリストに対応する")","}","]"を追加する.
- もし,Sを見ていく中で終わり括弧が含まれていたら,checkから取り除く.
- 対応する始まり括弧がない場合,処理終了.
